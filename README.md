# Library Manager – Python Mini Project

A simple CLI-based Library Inventory & Borrowing System built in Python for the Unit-2 Mini Project (ETCCCPP103 – Problem Solving with Python).

##  Features
- Add, view, and search books  
- Borrow and return books  
- Track borrowed records  
- Menu-driven program (loops, functions, dicts, list comprehensions)

##  Project Structure
library_manager/  
-- library.py  

##  How to Run
Open terminal in VS Code and run:

python library.py
